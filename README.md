# Student Grade Calculator

## Description
A simple Python project that calculates a student's total marks, average marks, and grade based on marks entered by the user.

## Features
- Takes student name as input
- Accepts marks for Maths, Science, and English
- Calculates total marks
- Calculates average marks
- Assigns grades (A+, A, B, C, or Fail)

## Technologies Used
- Python 3

## Concepts Used
- Variables
- User Input
- Arithmetic Operators
- Conditional Statements (if-elif-else)
